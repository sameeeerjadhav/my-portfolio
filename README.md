# Sameer's Portfolio 🚀

Welcome to my personal portfolio repository! This project is a uniquely designed, interactive, neo-brutalist portfolio built to showcase my experience, projects, skills, and technical background.

## 🎨 Features
- **Neo-Brutalism Design:** High contrast colors, bold borders, and distinct shadows for a unique, modern aesthetic.
- **SameerOS Terminal:** A fully functional interactive mini-terminal with fun hidden commands (`matrix`, `fetch`, `hack`) and mini-games.
- **Retro OS Client Explorer:** A fully styled "Windows 95/Retro OS" style file explorer window specifically built to showcase past client projects.
- **Doodle Blackboard:** An interactive HTML5 Canvas board in the Education section where visitors can draw and leave doodles.
- **Fully Responsive:** Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack
- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS (v4)
- **Icons:** React Icons

## 🚀 Getting Started

To run this project locally, simply follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository
```bash
git clone https://github.com/sameeeerjadhav/my-portfolio.git
cd my-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`
